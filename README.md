# Mpesa Daraja STK-Push Nodejs App
This app implements the Online lipa na mpesa api

#Prerequisites
Node v6+, 8+ recommended.
ExpressJs
ES6 knowledge
Api Key and Secret from https://developer.safaricom.co.ke/home
