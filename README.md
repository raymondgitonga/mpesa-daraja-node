# Mpesa Daraja STK-Push Nodejs App
This app implements the Online lipa na mpesa api

# Prerequisites
1. Node v6+, 8+ recommended.
2. ExpressJs
3. ES6 knowledge
4. Api Key and Secret from https://developer.safaricom.co.ke/home

# Installation
1. npm i mpesa-online 
2. npm i express
